## Cursor - AI 代码编辑器

Cursor 是基于 VSCode 的 AI 代码编辑器，集成了强大的 AI 功能，让编程变得更加高效。

### 核心特性

#### 1. AI 代码补全
- **自动补全**：基于上下文的智能代码建议
- **多行补全**：一次性生成多行代码
- **跨文件理解**：理解整个项目的代码结构

#### 2. AI 聊天功能
- **代码解释**：解释代码的功能和逻辑
- **代码重构**：帮助重构和优化代码
- **问题解答**：回答编程相关问题
- **代码生成**：根据需求生成完整功能

#### 3. 代码编辑功能
- **Composer**：多文件编辑模式
- **代码审查**：AI 辅助代码审查
- **错误修复**：自动检测和修复常见错误

### 安装和配置

#### 系统要求
- Windows 10/11、macOS 10.15+、Linux
- 至少 4GB RAM（推荐 8GB+）
- 网络连接（用于 AI 功能）

#### 安装步骤

1. **下载安装包**
   - 访问 [Cursor 官网](https://cursor.sh)
   - 下载对应操作系统的安装包

2. **安装**
   - Windows：运行 `.exe` 安装程序
   - macOS：拖拽到 Applications 文件夹
   - Linux：解压并运行

3. **首次启动**
   - 创建账户或登录
   - 选择 AI 模型（GPT-4、Claude 等）
   - 配置 API 密钥（如需要）

#### 配置设置

**基本设置**
```json
{
  "cursor.ai.enabled": true,
  "cursor.ai.model": "gpt-4",
  "cursor.ai.maxTokens": 2000
}
```

**快捷键设置**
- `Ctrl/Cmd + K`：AI 编辑
- `Ctrl/Cmd + L`：打开 AI 聊天
- `Ctrl/Cmd + I`：内联编辑

### 使用技巧

#### 1. 高效使用 AI 补全

**提供清晰上下文**
- 在注释中描述需求
- 使用有意义的变量名
- 保持代码结构清晰

**示例**：
```python
# 创建一个函数，接收用户列表，返回按年龄排序的用户
def sort_users_by_age(users):
    # AI 会自动补全排序逻辑
```

#### 2. 使用 Composer 进行多文件编辑

**步骤**：
1. 打开 Composer（`Ctrl/Cmd + I`）
2. 描述要修改的功能
3. AI 会分析相关文件并生成修改建议
4. 审查并应用更改

**示例场景**：
- "添加用户认证功能"
- "重构这个模块，提高可测试性"
- "添加错误处理和日志记录"

#### 3. AI 聊天最佳实践

**明确的问题**
- ❌ "这个代码有问题"
- ✅ "这个函数为什么返回 None？如何修复？"

**提供上下文**
- 选中相关代码
- 描述具体问题
- 说明期望的结果

#### 4. 代码审查

**使用 AI 审查代码**
1. 选中要审查的代码
2. 打开 AI 聊天
3. 输入："请审查这段代码，找出潜在问题"

**常见审查点**
- 安全性问题
- 性能优化
- 代码风格
- 最佳实践

### 高级功能

#### 1. 自定义提示词

创建 `.cursorrules` 文件：
```
- 使用 TypeScript 严格模式
- 遵循 Airbnb 代码风格
- 优先使用函数式编程
- 添加详细的 JSDoc 注释
```

#### 2. 项目级配置

在项目根目录创建 `cursor.config.json`：
```json
{
  "model": "gpt-4",
  "temperature": 0.7,
  "maxTokens": 4000,
  "rules": [
    "使用项目特定的代码风格",
    "遵循项目的架构模式"
  ]
}
```

#### 3. 代码库索引

**启用代码库索引**
- 帮助 AI 理解整个项目
- 提高代码补全准确性
- 支持跨文件引用

**配置**：
1. 打开设置
2. 启用 "Index codebase"
3. 选择要索引的文件类型

### 常见问题

#### Q: Cursor 和 VSCode 有什么区别？
A: Cursor 基于 VSCode，但集成了强大的 AI 功能，包括代码补全、聊天、多文件编辑等。

#### Q: 需要付费吗？
A: Cursor 提供免费版和付费版。免费版有使用限制，付费版提供更多功能和更高的使用配额。

#### Q: 支持哪些编程语言？
A: 支持所有主流编程语言，包括 Python、JavaScript、TypeScript、Go、Rust 等。

#### Q: 数据隐私如何？
A: Cursor 提供本地模式，代码不会发送到服务器。但 AI 功能需要网络连接。

### 最佳实践

1. **明确需求**：在注释或聊天中清晰描述需求
2. **审查代码**：始终审查 AI 生成的代码
3. **逐步迭代**：先实现核心功能，再逐步完善
4. **学习模式**：理解 AI 生成的代码，不要盲目接受
5. **版本控制**：使用 Git 管理代码变更
