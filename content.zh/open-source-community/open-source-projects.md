## AI 编程相关开源项目

### 代码生成工具

#### Continue
- **描述**：开源的 AI 代码编辑器扩展
- **特点**：支持多种模型，可本地部署
- **GitHub**：https://github.com/continuedev/continue
- **适用**：需要自定义 AI 助手的开发者

#### Codeium（开源版本）
- **描述**：开源代码补全工具
- **特点**：免费，功能强大
- **GitHub**：https://github.com/Exafunction/codeium.vim
- **适用**：Vim/Neovim 用户

#### Aider
- **描述**：命令行 AI 编程助手
- **特点**：Git 集成，代码审查
- **GitHub**：https://github.com/paul-gauthier/aider
- **适用**：命令行爱好者

### 大模型相关

#### Ollama
- **描述**：本地运行大模型的工具
- **特点**：简单易用，支持多种模型
- **GitHub**：https://github.com/ollama/ollama
- **适用**：需要本地部署模型的开发者

#### LM Studio
- **描述**：本地 LLM 运行和管理工具
- **特点**：图形界面，易于使用
- **GitHub**：https://github.com/lmstudio-ai/lmstudio
- **适用**：不熟悉命令行的用户

#### Text Generation WebUI
- **描述**：Web 界面的文本生成工具
- **特点**：支持多种模型，功能丰富
- **GitHub**：https://github.com/oobabooga/text-generation-webui
- **适用**：需要 Web 界面的用户

### 贡献指南

#### 如何贡献

1. **Fork 项目**
   ```bash
   git clone https://github.com/username/project.git
   cd project
   ```

2. **创建分支**
   ```bash
   git checkout -b feature/your-feature
   ```

3. **提交更改**
   ```bash
   git commit -m "Add your feature"
   git push origin feature/your-feature
   ```

4. **创建 PR**
   - 在 GitHub 上创建 Pull Request
   - 描述更改内容
   - 等待审查

#### 贡献类型

- **代码贡献**：新功能、Bug 修复
- **文档贡献**：改进文档、翻译
- **测试贡献**：添加测试用例
- **社区贡献**：回答问题、帮助新用户
